## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/NabilIsaac/react-native-expo.git
cd react-native-expo
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npx expo start
```
